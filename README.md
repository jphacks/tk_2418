## 製品概要
### 背景(製品開発のきっかけ、課題等）
近年、日本では研究者不足が深刻な問題となっていますが、一方で就職活動の早期化が進行しています。その結果、大学生が研究に専念するハードルが高くなっているのが現状です。
### 製品説明（具体的な製品の説明）
学生が積極的に研究活動に取り組めるような、研究をサポートするSNSモバイルコミュニティアプリです。
